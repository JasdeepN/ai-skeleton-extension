# Startup Prompt

**⚠️ CRITICAL: THIS FILE IS READ-ONLY ⚠️**
**DO NOT MODIFY THIS PROMPT FILE. It is a template for agent workflows.**
**All work, plans, and context must be saved to AI-Memory/, NOT here.**


[New Task Or Continuing Session?]

## Instructions for Every New Chat

- Ask whether this is a new task or continuation of a previous session.
- Check if AI-Memory is active, if not activate it.
- Check if MCPs are enabled. if not, enable them.

## New Task Checklist
- Read current project context, current archtechture from memory bank.
- Clear current context for any ongoing tasks or blockers.
- Print "Ready for new instructions."

## Continuing Session Checklist
- Read current project context from memory bank.
- Check current context for any ongoing tasks or blockers.
- Briefly state your main goal or question for this session.
- Mention what you last worked on or any blockers you encountered.

**Tip:** You can always ask for a summary of the current context, memory, or workflow at any time.

---

This prompt is automatically shown at the start of every new chat to help you get oriented and productive quickly.
